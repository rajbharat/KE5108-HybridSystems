Age -> Income_Education Weighting with range (-0.5, 0.5)¶

This weighting will be used to effect the rule given "Education – a higher level is better. Education is more important for middle-aged customers. For older customers income is more important than education-level"

import numpy as np

import skfuzzy as fuzz

# prior to this, need to scale age to 0-10
